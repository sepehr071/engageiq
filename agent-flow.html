Tool Reference

Tools the LLM can call during conversation

Main Agent Tools

detect_visitor_role(role)

Capture visitor's job role for personalization

Param: role (string) - visitor's job title

present_engageiq()

Present EngageIQ product with CORE/DFKI client images

Sends images to frontend via products topic

+2 Intent

collect_challenge(challenge)

Store visitor's biggest challenge answer

Param: challenge (string)

+3 specific / +1 vague

check_intent_and_proceed()

Check engagement level and determine next step

Score ≥3: offer lead capture

Score <3: try re-engage, then goodbye

connect_to_lead_capture(confirm)

Handoff to LeadCaptureAgent or say goodbye

Param: confirm (bool)

true → seamless handoff

false → save history + webhook

Lead Capture Tools

collect_lead_info(name, email, ...)

Save lead and send notifications

Params: name, email, company, role, phone

1. Validate email

2. Save JSON to leads/

3. Email to info@ayand.ai

4. Send webhook

start_new_conversation()

Restart conversation with fresh agent

1. Send {"clean": true}

2. Save history

3. Send webhook

4. Return new EngageIQAssistant

Real-World Conversation Trace

Example: Successful lead capture (Marketing role)

# Actor Message / Action State Change
1 Visitor Scans QR code, joins LiveKit room session_id = "user-xxx"
2 Agent "Welcome to Ayand AI! What brings you to our booth today?" first_message = true
3 Visitor "I work in marketing, just curious what you do." -
4 System Calls detect_visitor_role("marketing") visitor_role = "marketing"
5 System Calls present_engageiq() - sends CORE/DFKI images intent_score = 2
6 Agent "For marketing, EngageIQ shows which campaigns drive real demand. Which channels generate qualified demand for you?" -
7 Visitor "Honestly, we don't know. Facebook seems okay but hard to tell." -
8 System Calls collect_challenge() - detects vague answer intent_score = 3
biggest_challenge = "..."
9 System Calls check_intent_and_proceed() - Score ≥ 3, GOOD_SIGNAL qualified = true
10 Agent "Would you like me to send you more information?" -
11 Visitor "Sure, that sounds helpful." -
12 System Calls connect_to_lead_capture(true) HANDOFF to LeadCaptureAgent
13 LeadAgent "Great! Let me get your details. What's your name?" on_enter() triggers
14 Visitor "Anna Schmidt" -
15 LeadAgent "Thanks Anna. What's your email?" -
16 Visitor "anna@company.de" -
17 LeadAgent "Which company?" -
18 Visitor "RetailTech GmbH" -
19 System Calls collect_lead_info(name, email, company) email validated
lead saved to JSON
email sent to info@ayand.ai
webhook sent
20 LeadAgent "Thanks Anna! Team will be in touch. Have a great EuroShop!" "New Conversation" button sent
21 Visitor Clicks "New Conversation" -
22 System Calls start_new_conversation() {"clean: true} sent
history saved
webhook sent
NEW EngageIQAssistant

Frontend Protocol

Topic Payload When Sent
